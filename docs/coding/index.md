# Programming & Systems

*——building from the metal up* 🦀

Programming is more than just writing logic; it is about understanding how data flows through silicon. My focus lies in **Low-Level Systems**, **Hardware Architecture**, and **High-Performance Computing**, avoiding the "black box" approach of modern high-level frameworks.

## 📝 Current Tech Stack

- [x] **C/C++**: The "Mother Tongue" for systems and performance-critical 3D Vision.
- [ ] **Verilog**: Designing logic at the gate level (Hardware Architecture).
- [x] **Python**: Essential for scripting and Deep Learning research.
- [ ] **Assembly (x86/RISC-V)**: Understanding the ISA layer.

------

## 🛠️ Development Philosophy

## 1. No Black Boxes

I prefer manual memory management over Garbage Collection. Understanding `malloc` and `free` is non-negotiable for anyone serious about system performance.

## 2. Efficiency Over "Business Logic"

I find more satisfaction in optimizing a **Spatial Partitioning Tree** or a **Gaussian Splatting Renderer** than in building repetitive CRUD applications.

## 3. The Hardware-Software Interface

Bridging the gap between high-level algorithms and the actual hardware (CPU/GPU) is where the real engineering happens.
