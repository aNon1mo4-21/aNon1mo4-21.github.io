    Software & Tools: The Digital Workshop

    ——crafting an environment for maximum friction-less focus 🛠️

    A researcher is only as good as their tools. I prioritize tools that are lightweight, customizable, and keyboard-centric, allowing me to bridge the gap between abstract thought and technical execution.

    📝 Current Workflow To-Do

    • Dotfiles Management: Version controlling my system configurations via Git.
    • Neovim Mastery: Transitioning from VS Code for terminal-based C/C++ development.
    • Docker Containers: Isolation for reproducible Deep Learning environments.
    • Markdown/LaTeX: Streamlining the "Note-to-Publication" pipeline.

    💻 OS & Desktop Environment

    🐧 Linux (WSL2 / Ubuntu)

    The native habitat for Computer Vision and Systems Programming.

    • Shell: zsh with Oh My Zsh for a clean, informative prompt.
    • Terminal: Windows Terminal (customized) or Alacritty.
    • Multiplexer: tmux for persistent sessions during long model training.

    ✍️ Code & Document Editors

    🧩 VS Code

    My primary IDE for complex C++ projects and Python research.

    • Essential Extensions: C/C++ (IntelliSense), Pylance, LaTeX Workshop, and GitHub Copilot.
    • Vim Mode: Enabled. Navigating code without a mouse is non-negotiable.

    🖋️ Obsidian / MkDocs

    My "Second Brain."

    • Obsidian: For fleeting notes and daily logs.
    • MkDocs: The engine powering Abby's Wonderland (this site), converting technical notes into a structured digital garden.

    🔬 Research & Data Stack

    Category Tool Why?
    VC Git / GitHub The standard for collaboration and versioning.
    Math WolframAlpha Quick verification of integrals and matrix props.
    Viz MeshLab Inspecting 3D point clouds and Gaussian Splats.
    Docs Overleaf For collaborative \(\LaTeX\) writing.

    📂 Software Recommendations

    📦 Development

    • gcc / gdb: The fundamental toolchain.
    • CMake: The only way to manage C++ builds sanely.
    • Conda: Keeping Python environments from becoming a dependency hell.

    🎨 Design

    • Desmos: For rapid mathematical visualization.
    • Figma: Occasionally used for poster layouts and UI prototyping.
