    Linear Algebra: The Geometry of Data

    "The purely formal handle on algebra is a bit like a dry bone; the geometric intuition is the meat." — Adapted from 3Blue1Brown

    Linear Algebra is the engine behind Computer Vision and Spatial Intelligence. Following the "Axler Way" (LADR), I focus on linear maps and operators rather than just pushing numbers around in matrices.

    🎯 Current Focus (LADR Pathway)

    • Vector Spaces & Subspaces: Understanding the "canvas" where linear transformations live.
    • Linear Maps: Treating matrices as representations of functions between vector spaces.
    • Eigenvalues & Eigenvectors: Finding the invariant directions of a transformation.
    • Inner Product Spaces: The foundation of geometry—orthogonality, projections, and the Adjoint operator.

    🔍 3D Vision Applications

    • Change of Basis: Crucial for transforming coordinates between Camera Space, World Space, and Screen Space.
    • Singular Value Decomposition (SVD): The "Swiss Army Knife" for solving Overdetermined Systems and Essential Matrix estimation.
    • Projective Geometry: Understanding how 3D points map to 2D planes via Homographies.

    📝 Key Insights

    • Determinants are Volume: A determinant isn't just a formula; it's the scaling factor of a transformation's volume.
    • Spectral Theorem: Why symmetric matrices (like Covariance matrices) are so well-behaved and useful for PCA.

    🛠️ Working Sandbox

    • Implement basic Matrix Decomposition in pure C (for the "Low-level" soul).
    • Visualize Span and Linear Independence using Manim/Python.
    • Master the Gram-Schmidt Process for orthonormalization.
